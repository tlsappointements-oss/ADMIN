Setup Guide
Link your Gmail in 3 minutes using Google Apps Script — exactly like your existing visa platform
STEP 1

Create your Google Apps Script

Go to script.google.com → New Project → paste this exact code → Save

1a. Open Google Apps Script

Log into the Gmail account you want to send FROM, then go to script.google.com and create a new project.

1b. Paste this script (replaces Code.gs)

Delete all existing code and paste the script below:

// CareerBlast — Google Apps Script Email Sender // Paste this into script.google.com → Code.gs function doPost(e) { var data = JSON.parse(e.postData.contents); var to = data.to; var subject = data.subject; var html = data.html; var name = data.name || to; try { GmailApp.sendEmail(to, subject, '', { htmlBody: html, name: name }); return ContentService .createTextOutput(JSON.stringify({status:'sent'})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({status:'error', message: err.toString()})) .setMimeType(ContentService.MimeType.JSON); } } // Test function — run this first to authorize function testSend() { GmailApp.sendEmail('your-email@gmail.com', 'CareerBlast Test', '', { htmlBody: '<p>CareerBlast is connected! ✅</p>', name: 'CareerBlast' }); }

1c. Deploy as Web App

Click Deploy → New Deployment → Web App
— Execute as: Me
— Who has access: Anyone
— Click Deploy → authorize → copy the URL

STEP 2 — Paste your GAS URL here
STEP 3 — Firebase (already configured)
✅ Firebase is pre-configured with your project cars-website-558c0
Contacts and email logs are stored automatically in Firestore.
⚠️ Important Notes
  • Gmail free accounts can send ~500 emails/day — Google Workspace allows 2,000/day
  • Always run the testSend() function first in GAS to authorize permissions
  • The GAS script sends from whichever Gmail is logged in when you deploy
  • Re-deploy after any code change to get a new URL
  • Keep emails personalized — avoid spammy subject lines
