Setup Guide
Fix the Firebase error and link your Gmail — 5 minutes total
⚠️ Fix "Missing or insufficient permissions" first:
This error happens because Firestore blocks all writes unless you're authenticated AND the security rules allow it.
Go to Firebase Console → Firestore Database → Rules tab and replace everything with the rules below.
FIRESTORE RULES — Firebase Console → Firestore → Rules → Publish
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { // Each user only accesses their own subcollection match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }
Also enable Firebase Authentication
  1. Go to Firebase Console → Authentication → Sign-in method
  2. Enable Email/Password
  3. Enable Google (for Google sign-in)
  4. Add your domain to Authorized domains if deploying online
GOOGLE APPS SCRIPT — Link your Gmail

Create GAS Script (3 minutes)

Log into the Gmail you want to send FROM → script.google.com → New Project → paste below

Step 1 — Open script.google.com (logged into your sending Gmail)

Create a new project. Delete all existing code in Code.gs.

Step 2 — Paste this script

// CareerBlast GAS Sender — paste in script.google.com → Code.gs function doPost(e) { var data = JSON.parse(e.postData.contents); var opts = { htmlBody: data.html, name: data.name || data.to }; // Attach resume if sent as base64 if (data.attachment) { var bytes = Utilities.base64Decode(data.attachment.data); var blob = Utilities.newBlob( bytes, data.attachment.mimeType, data.attachment.filename ); opts.attachments = [blob]; } try { GmailApp.sendEmail(data.to, data.subject, '', opts); return ContentService .createTextOutput(JSON.stringify({ status: 'sent' })) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ status: 'error', message: err.toString() })) .setMimeType(ContentService.MimeType.JSON); } } // Run this ONCE first to grant Gmail permission function testSend() { GmailApp.sendEmail( 'your-email@gmail.com', // ← change this to your email 'CareerBlast Connected ✅', '', { htmlBody: '<p>Your CareerBlast is linked!</p>', name: 'CareerBlast' } ); }

Step 3 — Run testSend() first (IMPORTANT)

Select testSend from the function dropdown → click ▶ Run → click Allow when permission dialog appears. Without this, emails won't send.

Step 4 — Deploy → New Deployment → Web App

Execute as: Me  ·  Who has access: Anyone  ·  Click Deploy → copy the long /exec URL

Step 3 — Paste your GAS URL
⚠️ Notes
  • Gmail free accounts: ~500 emails/day · Google Workspace: 2,000/day
  • Always run testSend() first to grant Gmail permission
  • GAS sends from whichever Gmail was logged in when you deployed
  • Re-deploy after any code change to get a new URL
  • Resume is base64-encoded and sent as attachment — keep under 5MB
